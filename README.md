🎓 EduInBridge

EduInBridge is an Android-based educational platform designed to connect students, mentors, and institutions through a single application. The platform provides learning resources, mentorship opportunities, curriculum management, doubt-solving features, and AI-powered academic assistance.

✨ Features

👨‍🎓 Student Features

- Secure Login & Registration
- Profile Management
- Learning Materials Access
- AI Chat Assistant
- Doubt Discussion System
- Curriculum Viewing
- Mentorship Programs

🏫 Institution Features

- Institution Dashboard
- Curriculum Management
- Mentor Management
- Student Monitoring

🤖 AI Features

- Gemini AI Integration
- AI-powered Learning Support
- Academic Guidance

🚀 Setup Instructions

1. Clone Repository

git clone https://github.com/Shreyas99k/EduInBridge.git

2. Import Database

1. Open phpMyAdmin
2. Create a database named:

eduinsight

3. Import the provided SQL file.

3. Configure Backend

1. Copy the backend folder to:

xampp/htdocs/

2. Start Apache and MySQL from XAMPP.

4. Configure Android App

Update the API URL inside "Config.java":

public static final String BASE_URL = "http://10.0.2.2/your_api_folder/";

5. Run Application

- Open project in Android Studio
- Sync Gradle
- Run on Emulator or Physical Device
